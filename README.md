这里是我对对spring学习的源码，每一个目录都对应是一个独立的模块，是一个maven项目，可以独立地导入IDEA里面运行。

* spring-IoC  
对Spring Ioc概念的实例讲解的demo
* Spring-AOP
对Spring AOP概念的讲解demo
* spring-tx  
对Spring+MyBatis组合使用事务的demo
* mybatis
对MyBatis使用的基本配置
* mybatis-settings
对MyBatis配置文件的详细讲解
* SSM_integrated   
对SSM整合开发基本配置的demo（里面包含两个模块，一个是使用xml配置的，一个是使用java配置的）
